# Introduction

Welcome to TatakaiCore documentation.

## Base URLs

- Compatibility API: `/api`
- Proxy API: `/api/proxy`
- V2 API: `/api/v2`

## Useful Links

- Human docs UI: `/docs`
- Full endpoint list: `/docs/endpoints`
- LLM docs text: `/api/v2/docs/llm`
- Machine endpoint JSON: `/api/v2/docs/endpoints-json`

## Health and Version

- `GET /health`
- `GET /v`

## Notes

- HiAnime routes are mounted under `/api/v2/hianime`.
- Legacy compatibility stream routes remain under `/api`.
